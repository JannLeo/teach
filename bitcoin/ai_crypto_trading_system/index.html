基于深度学习和智能决策的端到端自动化交易平台

项目概述

本项目基于深度研究报告《AI驱动加密货币自动化交易系统的可行性研究》， 成功实现了一个完整的、分层模块化的AI交易系统。系统采用管道-过滤器架构， 通过六大核心模块实现了从数据采集到交易执行的端到端自动化流程。

核心特性

  • AI驱动决策: 基于大型语言模型的智能交易决策
  • 模块化架构: 管道-过滤器架构，高内聚低耦合
  • 实时数据处理: 支持多源数据采集和实时特征工程
  • 智能风险管理: 动态风险评估和仓位管理
  • Web界面监控: 实时系统状态监控和性能分析

系统功能模块

数据采集模块

从交易所和GMGN等平台获取实时市场数据，支持K线、订单簿、交易量等多维度数据，集成聪明钱动向等链上信号。

  • 实时价格数据获取
  • K线历史数据
  • 订单簿深度数据
  • 聪明钱动向分析

特征工程模块

数据清洗和预处理，技术指标计算，市场情绪特征提取，为AI决策提供高质量的特征输入。

  • 价格特征提取
  • 技术指标计算
  • 市场情绪分析
  • 风险评分计算

AI决策模块

基于DeepSeek API的智能决策引擎，结构化输出和规则验证，置信度评估和决策解释。

  • LLM智能分析
  • 结构化输出
  • 置信度评估
  • 决策历史记录

风险管理模块

多维度风险评估和交易信号过滤，动态仓位管理和止损策略，熔断机制和亏损保护。

  • 多维度风险评估
  • 动态仓位管理
  • 熔断机制
  • 亏损保护

订单执行模块

将交易信号转化为实际市场交易，支持多个交易所的API集成，智能订单路由和执行监控。

  • 多交易所支持
  • 智能订单路由
  • 执行监控
  • 交易历史统计

监控与性能模块

系统监控、性能分析和实时状态展示，集成Web界面和实时数据推送。

  • 实时系统监控
  • 性能分析
  • Web界面展示
  • 实时警报系统

快速开始

安装和配置

1. 环境准备
# 克隆项目 git clone cd ai_crypto_trading_system # 创建虚拟环境 python3 -m venv venv source venv/bin/activate # Linux/Mac # 或 venv\Scripts\activate # Windows
2. 安装依赖
pip install -r requirements.txt
3. 配置系统
# 复制配置文件 cp config/example_config.yaml config/config.yaml # 编辑配置文件，设置API密钥等参数 nano config/config.yaml

启动系统

一键启动
./quick_start.sh
手动启动
# 启动完整系统 python run.py # 启动模拟交易模式 python run.py --mode paper # 仅启动Web界面 python run.py --web-only # 查看更多选项 python run.py --help
访问Web界面
# 打开浏览器访问 http://localhost:8081
